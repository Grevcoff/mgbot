# MicroGreen Telegram Bot

Telegram-бот для учёта микрозелени с использованием aiogram 3.x. Бот помогает управлять партиями, отслеживать этапы роста, вести учёт времени и организовывать продажи.

## 🌱 Возможности

- **🌱 Управление партиями**: Создание и отслеживание партий микрозелени
- **⏰ Учёт времени**: Автоматическое отслеживание этапов роста (замачивание, темнота, свет)
- **🔔 Уведомления**: Напоминания о завершении этапов с ручным подтверждением переходов
- **🛒 Продажа лотков**: Корзина для продажи с выбором конкретных лотков
- **📦 Шаблоны культур**: Настройка параметров для разных культур
- **📊 Статистика**: Общая статистика по проекту

## 🛠️ Технический стек

- **Python 3.10+**
- **aiogram 3.x** - фреймворк для Telegram ботов
- **aiosqlite** - асинхронная работа с SQLite
- **asyncio** - фоновые задачи и планировщик

## 📦 Установка и настройка

### 1. Клонирование и установка зависимостей

```bash
git clone <repository-url>
cd mgbot
pip install -r requirements.txt
```

### 2. Настройка конфигурации

Скопируйте и отредактируйте файл `config.py`:

```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"  # Токен вашего бота от @BotFather
ADMIN_ID = 123456789  # Ваш Telegram ID для административных функций
```

Или используйте переменные окружения:

```bash
export BOT_TOKEN="your_bot_token"
export ADMIN_ID="your_telegram_id"
```

### 3. Получение токена бота

1. Найдите [@BotFather](https://t.me/botfather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен

### 4. Получение Telegram ID

1. Найдите [@userinfobot](https://t.me/userinfobot) в Telegram
2. Отправьте любое сообщение
3. Скопируйте ваш ID

### 5. Запуск бота

```bash
python main.py
```

## 🗄️ Структура базы данных

Бот использует SQLite с следующими таблицами:

### varieties (Шаблоны культур)
- `id` - уникальный идентификатор
- `name` - название культуры
- `seeds_per_lot` - грамм семян на лоток
- `seed_cost_per_gram` - цена семян за грамм
- `base_cost` - базовая стоимость (контейнер + субстрат)
- `default_sale_price` - цена продажи по умолчанию
- `soak_hours` - часы замачивания
- `dark_hours` - часы в темноте
- `light_hours` - часы на свету
- `is_active` - флаг активности (мягкое удаление)

### batches (Партии)
- `id` - уникальный идентификатор
- `variety_id` - ID культуры
- `quantity` - количество лотков в партии
- `current_stage` - текущий этап ('soak', 'dark', 'light', 'ready')
- `stage_started_at` - время начала текущего этапа
- `notified` - флаг отправки уведомления

### lots (Лотки)
- `id` - уникальный идентификатор
- `batch_id` - ID партии
- `lot_code` - уникальный код (MG-YYYY-NNN)
- `status` - статус ('growing', 'sold', 'written_off')
- `snapshot_cost` - себестоимость на момент создания
- `sale_price` - цена продажи

### orders (Заказы)
- `id` - уникальный идентификатор
- `buyer_name` - имя покупателя
- `total_amount` - общая сумма
- `created_at` - дата создания

### order_items (Позиции заказа)
- `id` - уникальный идентификатор
- `order_id` - ID заказа
- `lot_id` - ID лотка
- `price_at_sale` - цена на момент продажи

## 🚀 Использование бота

### Главное меню

При запуске бот предлагает следующие опции:

- **📊 Статистика** - просмотр общей статистики проекта
- **🌱 Мои партии** - управление партиями
- **➕ Новая партия** - создание новой партии
- **🛒 Продажа лотков** - процесс продажи
- **📦 Шаблоны культур** - управление шаблонами
- **⚙️ Настройки** - административные функции

### Создание шаблона культуры

1. Перейдите в "📦 Шаблоны культур"
2. Нажмите "➕ Добавить шаблон"
3. Заполните 8 шагов:
   - Название культуры
   - Семян на лоток (грамм)
   - Цена семян за грамм
   - Базовая стоимость
   - Цена продажи по умолчанию
   - Часы замачивания
   - Часы в темноте
   - Часы на свету

### Создание партии

1. Перейдите в "➕ Новая партия"
2. Выберите культуру из списка
3. Укажите количество лотков
4. Бот автоматически создаст лоты с кодами MG-YYYY-NNN

### Отслеживание этапов

Бот автоматически проверяет завершение этапов каждые 15 минут и отправляет уведомления админу с кнопками:

- **✅ Перевести дальше** - перевод на следующий этап
- **⏸️ Напомнить через 1ч** - отложить уведомление

### Продажа лотков

1. Перейдите в "🛒 Продажа лотков"
2. Выберите партию с лотками на стадии "ready"
3. Выберите конкретные лоты нажатием на номера
4. Установите цену (стандартная или自定义)
5. Добавьте в корзину
6. Введите имя покупателя для оформления заказа

## 📝 Команды

- `/start` - запуск бота и показ главного меню
- `/reset_db` - сброс базы данных (только для админа)

## 🔧 Административные функции

Доступны пользователю с ID, указанным в `ADMIN_ID`:

- **🗑️ Сбросить БД** - полное удаление всех данных
- **📤 Экспорт данных** - просмотр статистики экспорта

## 🐛 Логирование

Бот ведёт логирование в файл `bot.log` и консоль. Уровень логирования настраивается в `config.py`.

## 🔄 Резервное копирование

База данных хранится в файле `microgreen_bot.db`. Рекомендуется регулярно создавать резервные копии этого файла.

## 📱 Поддержка

При возникновении проблем:

1. Проверьте логи в файле `bot.log`
2. Убедитесь, что токен бота корректен
3. Проверьте, что указан правильный ADMIN_ID
4. Убедитесь, что установлены все зависимости из `requirements.txt`
